    Hillary | Personal Portfolio Website

A dark, techy, and professional personal portfolio website built with vanilla HTML, CSS, and JavaScript. Designed to showcase my skills, projects, work experience, and services as a web developer and Information Systems graduate.

         Built With

- HTML5 — Semantic, accessible structure
- CSS3 — Custom properties, Flexbox, Grid, animations
- JavaScript (Vanilla) — Typing effect, scroll reveal, skill bar animations, mobile menu
- Google Fonts — Orbitron, Share Tech Mono, Inter
- Font Awesome — Icons throughout the site

    Features

- Responsive design — works on mobile, tablet, and desktop
- Animated glitch effect on hero name
- Typing animation cycling through roles
- Profile photo with "Available for work" badge
- Scroll-triggered skill bar animations
- Scroll reveal on all sections
- Active navigation link highlighting
- Mobile hamburger menu
- Experience timeline (education & work attachments)
- Project cards with live site and GitHub links
- Working contact form
- CV download button
- Dark & techy aesthetic with a clean grid background

    Getting Started

    1. Clone the repository

    bash
git clone https://github.com/your-github-username/portfolio.git
cd portfolio

    2. Add your assets

- Place your profile photo at `assets/profile.jpg`
- Place your CV at `assets/hillary-cv.pdf`

        3. Update your personal details

Open `index.html` and replace the following placeholders:

 Placeholder | Replace With 

 Your GitHub username 
 Your LinkedIn profile slug 
 Your actual email address 
 Your phone number 

    4. Open locally

Open `index.html` using Live Server in VS Code for a better development experience.
